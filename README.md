# NootraGenie

Your perfect nootropic stack, granted. A personalized nootropic recommendation tool.

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repo
3. Click Deploy — done

## Stack

- React 18
- Vite 5
- Zero external dependencies beyond React
